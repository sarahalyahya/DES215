Web Design is 95% Typography

Since almost all of the information on the web is written language, it is only logical that a web designer should get good training in the main discipline of shaping written information, in other words: Typography

Why Typography Matters

To be effective on a website, typography needs to be clean, clear, appealing, and readable. Good web typography can attract, engage, and keep your viewer on your site.

Using font properties allows us to have huge control when styling text.
